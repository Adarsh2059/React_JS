## рџ“љ Concepts Learned & Implemented

### 1. Custom Hooks

Created a custom hook (`useCurrencyInfo`) to fetch and manage currency exchange data efficiently.

### 2. Component Reusability

Built reusable components such as `InputBox` to keep the code modular, maintainable, and scalable.

### 3. Barrel Exports (index.js)

Used an `index.js` file to export components from a single location, resulting in cleaner and more organized imports.

### 4. API Integration

Integrated a live currency exchange API to fetch up-to-date conversion rates and perform accurate currency conversions.

### 5. React State Management

Utilized React Hooks (`useState` and `useEffect`) to manage application state and handle asynchronous data fetching.
